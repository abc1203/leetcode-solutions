List []: allows duplicates, mutable
  Methods:
     - len(), append(), insert(), remove()


Tuple (): similar to lists but immutable


Set {}: no duplicates, mutable, implemented using hash tables
  Methods:
     - to access 1st element of set: next(iter(set_name))
     - to remove element: set_name.remove(elem)


Dict {}: key-value pairs, mutable, implemented using hash tables
  Methods:
     - keys(), values(), items()
     - mp[key] = value
